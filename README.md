
                                                                     AI Study Tool

A computer-vision-based study program that uses your webcam to keep you in track. It tracks your eyes to prevent you from falling asleep and monitors your face position to fix your posture
I Used
Python 3.x
OpenCV (`cv2`): For webcam feed  and image processing.
Haar cascade: Pre-trained XML models for frontal face and eye detection.
Winsound: Built-in Python library for native Windows that alarm triggers.
